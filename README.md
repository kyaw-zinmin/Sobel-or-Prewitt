# Sobel or Prewitt Operator
#### Topic: Image Processing (CSC231)

A simple python script that takes an image and process it using Sobel or Prewitt Operator.

### Prerequisites
1. uv
[uv Installation Guide](https://docs.astral.sh/uv/getting-started/installation/)

### Get Started
1. Clone the repo and `cd` into the project
2. Open Terminal at project directory
3. Inside the terminal, `uv init`
4. Then, `uv add opencv-python matplotlib pyqt6`
5. After the packages have been installed, `uv run main.py`
